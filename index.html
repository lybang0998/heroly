About grayscale

Grayscale is a free Bootstrap 33 theme created by Start Bootstrap.It can be yours right noi, simply download the template on the preview page. The theme is open source, and you can use it for any purpose, personal or commercial.


This theme features stock photo by Gratisdography along with a custom Google maps skin courtesy of Snazzy Maps.


Grayscale includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization.
